# Cowork Skills

The `.skill` files for Claude desktop Cowork mode are not stored here because they are binary zip archives.

To install them, download from the releases page or rebuild them using the skill creator in Cowork mode.

## Skills

- **notification** — Sends a Telegram alert before requesting computer-use permissions
- **assistant** — Startup skill: sends Telegram notification and ensures daemon is running at the start of each project
